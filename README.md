# Virtualan CRUD Application

A full stack CRUD application built using React, FastAPI, and MongoDB Atlas following a Three Tier Architecture.

---

## Tech Stack

Frontend  
React (Vite)

Backend  
FastAPI

Database  
MongoDB Atlas

---

## Architecture

This project follows Three Tier Architecture.

Presentation Layer  
React user interface used to manage users.

Application / Logic Layer  
FastAPI REST APIs handle business logic.

Data Layer  
MongoDB Atlas database used to store user data.

---

## Features

Create User  
View Users  
Update User  
Delete User
